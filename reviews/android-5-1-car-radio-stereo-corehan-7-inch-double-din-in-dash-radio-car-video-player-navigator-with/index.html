excellent head unit

I really like this radio. It has a ton of great features and only a few drawbacks. The system itself runs quickly, boot up time is about ten seconds. I’ve noticed no lagging running apps. The screen resolution is excellent. The touch screen works very well too. It has the google play store pre-installed so you can install Google Maps, Gmail, Pandora etc. So far I’ve only found one app that wouldn’t install which was PowerAmp.Another great feature that I didn’t see in the description is you don’t need an adapter for the steering wheel controls. It is pre-wired for steering controls and has a nice app built in that lets you set it up.So the downsides… It is slightly larger than a standard double din radio by about 3⁄16” of an inch in width and height. It’s just enough to be a hassle. It took almost 3 hours for me to cut and sand the opening on my dash kit to make this fit and look good. The radio app starts up immediately by default which is annoying. It will start playing at what ever volume the system was shut off at and it can’t be turned down until the system completely boots. Took me a while to figure out how to disable this. All you have to do is hit the back button on the left hand side of the head unit and it will shut off the radio until you start the app again.I also wanted to let people know what the passwords were to get into the Android Developer options and the “Extra Settings” menus.Developer options- 7890Extra Settings- 123456update I’ve had the unit for a few months now and no major issues.
