    Product Description

This Fanmis bike computer is wireless and automatic awake. Different from other one, no manual operation needed during trip.

If stop riding and no signal inputted within 300 seconds, it will enter sleeping mode. Then all functions are off except clock for power saving, it is not completely close, so once you start riding again, it will automatic awake and turn back to working state.

If you won’t ride for a long time, get the battery out to close the computer unit completely and prevent it. And when start riding ggain, please reset the parameters.

This bike computer will make sports life more free and comfortable, so it is becoming more popular among riders.

Parameters should be setted after installing battery 1.Tyre circumference:(0mm~9999mm) 2.Speed Unit(km/h,mile/h) 3.The Last Value of Odometer 4.Clock Format (12H/24H)

Functions 1.SPD Current Speed 2.ODO Odometer (0.1～9999.9 Km or Mile） 3.DST Trip Distance 4.MAX Maximum Speed 5.AVS Average Speed 6.TM Trip Time 7.CLK Clock / (12H/24H) 8.Scan 9.“+”“-“Comparison 10.Freeze Frame Memory 11.Backlight 12.Signal Interference-resistance 13.Sleeping Mode 14.AUTO ON/OFF

Specifications Power source: CR2032 battery Dimensions: 11.5 * 7* 4cm Package weight: 76g

Package Includes 1 * Computer unit 1 * CR2032 battery 1 * Mounting bracket 1 * Magnet 1 * Sensor 6 * Cable tie 1 * O-Ring （for installation assist） 1 * User manual
